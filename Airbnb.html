# After encoding categorical variables
data_encoded = pd.get_dummies(data, columns=['room_type', 'cancellation_policy', 'cleaning_fee', 'instant_bookable'], drop_first=True)

# Select features and target

target = 'log_price'
X = data_encoded.drop('log_price',axis=1)
y = data_encoded[target]

# Fill missing values in X features (if any left) with median or mean
for col in X.columns:
    if X[col].isnull().any():
        # Use median for numeric features
        if X[col].dtype in [np.float64, np.int64]:
            X[col] = X[col].fillna(X[col].median())
        else:
            X[col] = X[col].fillna(X[col].mode()[0])  # For categorical, fill with mode

CONCLUSION AND RECOMMENDATIONS¶

Conclusions:-¶

Rental prices (log scale) increase with larger capacity, more bedrooms/bathrooms, and higher review scores. Entire homes/apartments and strict cancellation policies generally lead to higher prices.The model explains a moderate degree of variance. Some price drivers (e.g., location amenities) may not be captured but can be included in future analyses.

Recommendations:¶

Hosts should maximize guest capacity and maintain high review ratings for premium pricing.Consider offering entire home apartment listings and stricter policies for increased revenue. Airbnb can promote instant bookable and high-rated properties for both host satisfaction and business growth.
